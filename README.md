# ColumnSeis E.060 — Web App

Software de diseño sismorresistente de columnas. **100% en el navegador, sin backend.**

## Opción 1: Netlify (GRATIS, recomendado)
1. Crear cuenta en https://netlify.com (gratis)
2. Drag & drop de la carpeta `dist/` en https://app.netlify.com/drop
3. ¡Listo! Obtienes un link como https://columnseis.netlify.app

## Opción 2: Ejecutar local
npm install && npm run dev

## Opción 3: Build manual
npm install && npm run build
# La carpeta dist/ contiene la app lista para cualquier hosting

Fórmula Excel:

--- PERÍODO TL (Tabla N°5 - Art. 14) ---
Depende SOLO del tipo de suelo:
  S0 → TL = 3.0 s

  S2 → TL = 3.0 s

Fórmula Excel:

    IF(LEFT(B7,2)="S3",2.5,IF(LEFT(B7,2)="S4",2,3)))))

--- COEFICIENTE DE REDUCCIÓN R ---
  R = Ro × Ia × Ip
Fórmula Excel:
  =C9*C10*C11  (donde C9=Ro, C10=Ia, C11=Ip)

--- GRAVEDAD ---
  g = 9.81 m/s²  (constante)

--- SA MÁXIMO ---
  Sa_max = (Z × U × 2.5 × S / R) × g
  (Corresponde al plateau del espectro donde C=2.5)
Fórmula Excel:
  =(C6*C8*2.5*C7/B16)*B17

--- SA/g MÁXIMO ---
  Sa_g_max = Sa_max / g = Z × U × 2.5 × S / R
Fórmula Excel:
  =B20/B17


################################################################################
# SECCIÓN 3: FÓRMULA DEL FACTOR DE AMPLIFICACIÓN SÍSMICA C(T)
# Art. 18, Tabla N°6 de la NTE E.030-2025
################################################################################

El factor C depende del período T de la estructura y de los períodos TP y TL del suelo.

CUATRO CONDICIONES (Art. 18):

  Condición 1: T < 0.2×Tp
    C(T) = 1 + 7.5 × (T / Tp)
    [Rama ascendente lineal desde C=1.0 hasta C=2.5]

  Condición 2: 0.2×Tp ≤ T ≤ Tp
    C(T) = 2.5
    [Plateau - aceleración constante máxima]

  Condición 3: Tp < T < TL
    C(T) = 2.5 × (Tp / T)
    [Rama descendente hiperbólica - zona de velocidades]

  Condición 4: T ≥ TL
    C(T) = 2.5 × (Tp × TL / T²)
    [Rama descendente parabólica - zona de desplazamientos]

Fórmula Excel (celda B8 en adelante de hoja ESPECTRO):
  =IF(A8<0.2*$E$4, 1+7.5*(A8/$E$4),
    IF(A8<=$E$4, 2.5,
      IF(A8<$F$4, 2.5*($E$4/A8),
        2.5*($E$4*$F$4/(A8^2)))))

Donde:
  A8     = T (período en segundos)
  $E$4   = Tp (referencia a celda con valor de ENTRADA)
  $F$4   = TL (referencia a celda con valor de ENTRADA)

CASO ESPECIAL T=0:
  En T=0, la fórmula C(T)=1+7.5*(0/Tp)=1, lo cual es correcto.
  El Excel usa =1 directamente en la primera fila (B7).
  Sa(T=0) = Z×U×1×S/R × g


################################################################################
# SECCIÓN 4: ESPECTRO DE PSEUDO-ACELERACIONES Sa(T)
################################################################################

La aceleración espectral se calcula como:

  Sa(T) = (Z × U × C(T) × S / R) × g    [en m/s²]
  Sa/g(T) = Z × U × C(T) × S / R         [adimensional, para ETABS]

Fórmula Excel (columna C de hoja ESPECTRO):
  =($A$4*$B$4*B8*$C$4/$D$4)*$G$4

Donde:
  $A$4 = Z (factor de zona)
  $B$4 = U (factor de uso)
  B8   = C(T) (factor de amplificación para ese período)
  $C$4 = S (factor de suelo)
  $D$4 = R (coeficiente de reducción)
  $G$4 = g = 9.81 m/s²

Sa/g (columna D):
  =C8/$G$4

################################################################################
# SECCIÓN 13: INTEGRACIÓN CON MÓDULO DE IRREGULARIDADES
################################################################################

FLUJO DE DATOS ENTRE MÓDULOS:

Si la app de irregularidades ya calcula los factores Ia e Ip finales:
  1. El usuario completa la verificación de irregularidades → obtiene Ia, Ip
  2. Al ir al módulo de espectro, los campos Ia e Ip se PRE-LLENAN
     con los valores calculados por el módulo de irregularidades.
  3. El usuario puede SOBRESCRIBIR manualmente si lo desea.
  4. R se recalcula automáticamente: R = Ro × Ia × Ip

const SISTEMA_RO = {
  "Pórticos C.A.": 8, "Dual C.A.": 7, "Muros C.A.": 6,
  "EMDL": 3.5, "Albañilería": 3, "Acero SMF": 8, "Acero SCBF": 7,
};

const g = 9.81;  // m/s²

// === Función C(T) - Tabla N°6 ===

function calcularC(T, Tp, TL) {
  if (T < 0.2 * Tp) {
    return 1 + 7.5 * (T / Tp);           // Rama ascendente
  } else if (T <= Tp) {
    return 2.5;                            // Plateau
  } else if (T < TL) {
    return 2.5 * (Tp / T);                // Decaimiento 1/T
  } else {
    return 2.5 * (Tp * TL / (T * T));     // Decaimiento 1/T²
  }
}

// === Generar espectro completo ===

function generarEspectro(Z, U, S, Ro, Ia, Ip, Tp, TL) {
  const R = Ro * Ia * Ip;
  const espectro = [];

  for (let i = 0; i <= 200; i++) {
    const T = i * 0.02;                    // 0.00 a 4.00, paso 0.02
    const C = calcularC(T, Tp, TL);
    const Sa = (Z * U * C * S / R) * g;    // m/s²
    const SaG = Sa / g;                     // adimensional

    espectro.push({ T, C, Sa, SaG });
  }

  return espectro;
}

// === Exportar para ETABS ===

function exportarETABS(espectro, paso = 1) {
  // paso=1 → cada punto (0.02s), paso=5 → cada 0.10s
  let texto = "";
  for (let i = 0; i < espectro.length; i += paso) {
    const { T, SaG } = espectro[i];
    texto += T.toFixed(2) + "\t" + SaG.toFixed(5) + "\n";
  }
  return texto;
}
